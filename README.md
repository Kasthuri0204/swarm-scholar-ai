🧠 SwarmScholar AI

Multi-Agent AI Research Assistant Using Swarm Intelligence

SwarmScholar AI is a multi-agent research automation system that simulates a collaborative team of AI researchers. It combines Retrieval-Augmented Generation (RAG) with specialized AI agents to generate structured academic research outputs including literature reviews, critiques, debates, hypotheses, and final reports.

Instead of using a single LLM response, SwarmScholar AI orchestrates a swarm of intelligent agents, each responsible for a specific research function.

🚀 Features

✅ Multi-Agent Research Pipeline
✅ Retrieval-Augmented Generation (RAG) for academic papers
✅ Automated Literature Review
✅ Critical Analysis Generation
✅ AI-based Research Debate Simulation
✅ Hypothesis Generation
✅ Structured Final Report with Citations
✅ Modular & Extensible Architecture


🏗️ Project Architecture

The system follows a Swarm Intelligence Model, where each agent performs a dedicated task:

User Topic Input
        ↓
Field Detector Agent
        ↓
Paper Fetcher (RAG)
        ↓
Literature Review Agent
        ↓
Critic Agent
        ↓
Debate Agent
        ↓
Hypothesis Agent
        ↓
Report Agent
        ↓
Final Research Report (Saved Output)



🤖 Agents Overview

1️⃣ Field Detector Agent

Identifies the research domain (AI, Healthcare, IoT, etc.)

Adapts prompt templates based on field

2️⃣ Literature Agent

Reads retrieved research papers

Generates structured literature review

3️⃣ Critic Agent

Performs critical analysis

Identifies research gaps and limitations

4️⃣ Debate Agent

Simulates contrasting viewpoints

Strengthens analytical depth

5️⃣ Hypothesis Agent

Proposes research hypotheses

Suggests possible experimental directions

6️⃣ Report Agent

Compiles all outputs

Generates a final structured research document with citations
